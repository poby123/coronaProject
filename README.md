# ssuCoronaProject_Client
## Dependencies
- GTTS (https://pypi.org/project/gTTS/)
```
pip3 install gtts
```

- PyQt5 (https://pypi.org/project/PyQt5/)
```
pip3 install PyQt5
```

## Usage
- Install
```
git clone https://github.com/poby123/ssuCoronaProject_Client
```

- Execute
```
python3 main.py
```
